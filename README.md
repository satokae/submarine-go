# Submarine (Go version)

[satokae/submarine](https://github.com/satokae/submarine) (プライベートリポジトリ) の Go 言語版実装です。オリジナル版と同一のアルゴリズムで動作します。

## 実行方法

Go がインストールされている環境で、以下のコマンドを実行してください。

```bash
go run cmd/submarine-go/main.go
```

オリジナル版とシード値の互換性はありません。
